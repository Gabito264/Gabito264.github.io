REALIZAR UN PROYECTO

Para poder realizar el proyecto, se utilizarán diferentes herramientas de trabajo. Algunas de ellas son las siguientes:

SQL server

La base de datos contendrá, como dice su nombre, los datos de los sistemas que utilizará el proyecto. Con estos datos, se puede organizar que datos se van a ver en el la aplicación.

Para más información de la herramienta,aquí
Unity Game Engine

La aplicación se realizará por medio de Unity. Con este podemos programar la vista de varios modelos con el uso de la cámara del dispositivo.

Para más información de la herramienta, aquí
React js, u otro framework web de javascript

El uso de un framework de javascript nos va a permitir cargar la aplicación de unity a la página, por lo que se podrá utilizar en todo momento, mientras el servidor lo permita de manera fácil y rápida.

Para más información de la herramienta, aquí
